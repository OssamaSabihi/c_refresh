# 0x0E. C - Structures, typedef

## Description
This project covers the introduction of user-defined data structures () and type aliasing () in low-level C development. It teaches how to group multiple variables of different types under a single unified memory block, copy dynamic internal pointers securely, and track manual custom struct layout constraints safely.

## Learning Objectives
* What structures are and why they are used.
* The difference between dot  and arrow  member selection operators.
* How to use  to create alias abstractions for existing types.
* How to safely handle internal deep string memory allocation configurations across dynamic tracking scopes.
